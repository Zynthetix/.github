## Projects

### [Zynthetix Voice](https://github.com/Zynthetix/zynthetix-voice)
Global hotkey speech-to-text for macOS. Double-tap right ⌥ to dictate into any app.  
Runs fully offline via **whisper.cpp** with Metal GPU acceleration — no API key, no internet after first model download.

```
Double-tap Right ⌥ → speak → text inserted wherever your cursor is.
```

**[⬇ Download v2.0.2 DMG](https://github.com/Zynthetix/zynthetix-voice/releases/latest)** — Apple Silicon required

---

### [Zynthetix AutoAgent](https://github.com/Zynthetix/zynthetix-autoagent)
Multi-agent AI workspace with terminal, kanban, context panel, and git worktree management.  
Built in Tauri + React. Designed for focused, distraction-free engineering sessions.

## Design System

All Zynthetix products follow a unified design language:

- **Dark-only** — deep charcoal surfaces, no light mode
- **Single accent** — `#4F76F6`, used precisely and sparingly
- **No gradients, no shadows, no glassmorphism**
- **8px spacing grid** — generous breathing room, never dense
- **Inter / SF Pro** — system-native typography, no decorative fonts
- **180–220ms transitions** — functional motion only

The full spec lives in [zynthetix-assets/THEME.md](https://github.com/Zynthetix/zynthetix-assets/blob/main/THEME.md).
